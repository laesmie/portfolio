REVIEW

  • Think first method is best
  • - Think, Design, Test, Create
  • HTML - If you've seen a picture, a button, some text, or watched a video, HTML was used to show it
  • CSS - If you've seen pages that weren't completely black and white, CSS was used to style it
  • JavaScript - If you've been on a website, clicked on something, and something happened (something changed), JavaScript was used
  • All HTML code goes between the html tags
  • All JavaScript code goes between the script tags
  • All CSS code goes between the style tags

Section 2.3 Notes

  • When you write an application you are communicating with the user - true communication is bidirectional
  • "Computers are stupid - Sometimes they can seem smart because they are so fast at beingn stupid"
  • Computers can't remember anything
  • Information stored or shared with the user is called output
  • Anything coming in from where it was stored, or anything the user enters, is called input
  • Input and Output is called IO - Input-Output
  • "Placeholder" is a property specific to textarea's and input tags - used as hints for the users - tell them what they should do - will go away when they start typing
  • h1 is the biggest header format tag - smallest is h4
  • Section tag fills across the entire width of the screen
  • var = variable - it is like box you can put stuff in, and it has a label - you decide the label - just make it match what it does
  • key-value pairs = description-stuff pairs - ex: phone book: key = name, value = phone number & address
